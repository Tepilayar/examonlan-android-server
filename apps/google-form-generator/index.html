Panduan Persiapan Soal Bergambar

Jika bank soal Word (.docx) Anda memiliki gambar, ikuti persiapan berikut sebelum membuat Google Form:

  1. Sediakan 2 file: File utama soal.docx (untuk diunggah di atas) dan salinannya soal.zip (diubah ekstensinya dari `.docx` ke `.zip`).
  2. Ekstrak gambar: Ekstrak file soal.zip tersebut, lalu buka folder word > media. Di dalamnya terdapat semua gambar dari file Word Anda.
  3. Beri nama sesuai nomor soal: Rename file gambar tersebut agar sesuai dengan nomor soalnya (misal: gambar untuk soal nomor 5 dinamai 5.png atau 5.jpg).
  4. Unggah ke Google Drive: Buat folder baru di Google Drive Anda, lalu unggah semua gambar yang sudah diganti namanya tadi ke dalamnya (jangan di-zip kembali, unggah sebagai file gambar satuan biasa).
  5. Salin & Bagikan Link Folder: Klik kanan folder tersebut di Google Drive, pilih Bagikan (Share), ubah Akses Umum menjadi Siapa saja yang memiliki link (Anyone with the link), klik Salin Link, lalu tempelkan link tersebut di langkah berikutnya (Edit & Kunci).

Punya file .doc atau .rtf?

Format .doc (Word lama) dan .rtf tidak dapat dibaca langsung oleh browser demi keamanan sistem.

Solusi termudah:

  1. Buka dokumen Anda di Microsoft Word atau Google Docs.
  2. Pilih menu File > Save As (Simpan Sebagai).
  3. Pilih format Word Document (.docx) dan simpan.
  4. Gunakan file .docx hasil konversi tersebut di halaman ini.
