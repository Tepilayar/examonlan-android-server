Cara Menggunakan File .seb

  1. Unduh Konfigurasi: Isi formulir di sebelah kiri dan klik tombol Download File Config SEB untuk mendapatkan file `.seb` Anda.
  2. Pasang Safe Exam Browser: Pastikan siswa telah mengunduh SEB pada perangkat mereka (iOS di App Store atau macOS/Windows di safeexambrowser.org).
  3. Buka File Konfigurasi:
    • Untuk iOS/iPhone: Kirim file `.seb` ini via WhatsApp/Email ke siswa. Siswa cukup mengetuk file tersebut dan memilih "Buka di Safe Exam Browser".
    • Untuk Windows/macOS: Siswa cukup melakukan klik ganda (double-click) pada file `.seb` yang telah Anda bagikan untuk membuka SEB dengan konfigurasi ujian.
  4. Ujian Terkunci: SEB akan otomatis mengunci perangkat siswa, membuka URL ujian Anda secara fullscreen, dan hanya mengizinkan keluar jika memasukkan PIN yang benar.
