Aplikasi Nomor Antrian

Sistem panggilan antrean offline-first yang handal, dirancang khusus untuk Sekolah, Puskesmas, dan Kantor Dinas. Menggunakan printer kasir thermal 58mm dan PC/Laptop biasa.

Fitur Utama Sistem Antrian

Empat modul utama dalam satu aplikasi yang saling terhubung secara real-time.

Layar Cetak Kiosk (Ambil Tiket)

Memungkinkan pengunjung mengambil nomor antrean secara mandiri melalui tombol-tombol berukuran besar pada layar sentuh. Desain tiket disesuaikan khusus untuk menghemat kertas kasir thermal ukuran 58mm.

  • Cetak instan & otomatis
  • Mendukung printer thermal 58mm
  • Kategori antrean dinamis

Layar Utama Display (Panggilan)

Menampilkan nomor antrean yang aktif dipanggil secara raksasa untuk TV/Monitor ruang tunggu. Dilengkapi dengan bunyi bel Ding-Dong modern dan panggilan suara otomatis dalam bahasa Indonesia yang alami.

  • Bel Ding-Dong modern
  • Voice Synthesizer (TTS) Indonesia
  • Dilengkapi teks pengumuman berjalan

Aplikasi Kontrol Loket (Petugas)

Antarmuka pemanggilan bagi petugas loket/teller untuk memanggil antrean berikutnya, memanggil ulang (recall), atau memindahkan kategori. Menampilkan estimasi antrean yang menunggu secara real-time.

  • Panggil Berikutnya & Panggil Ulang
  • Multi-loket (1-5 atau lebih)
  • Monitoring antrean tunggu

Dashboard Admin & Statistik

Laporan performa harian loket, grafik kunjungan per jam, dan rata-rata durasi tunggu pengunjung. Admin dapat mereset antrean, mengelola kategori layanan, dan mengekspor log panggilan detail ke CSV.

  • GrafikSVG interaktif
  • Ekspor CSV/Excel sekali klik
  • Pengaturan instansi & running text

Spesifikasi & Perangkat Pendukung

Kebutuhan minimal perangkat keras untuk menjalankan aplikasi secara lancar.

Komputer / Laptop Server

Menggunakan PC atau Laptop dengan sistem operasi Windows 7/8/10/11, macOS, atau Linux. RAM minimal 2GB dan penyimpanan yang cukup untuk menyimpan database riwayat antrean.

Printer Kasir Thermal 58mm

Mendukung printer kasir thermal standar ukuran lebar kertas 58mm (koneksi USB, LAN, atau Bluetooth). Tiket antrean akan tercetak secara instan dan rapi.

Browser & Layar Eksternal

Google Chrome atau Microsoft Edge untuk mendukung fitur panggilan suara (Web Speech API) dan pencetakan otomatis (silent printing) tanpa menampilkan dialog browser.

Panduan Setup Cepat

Ikuti langkah-langkah berikut untuk mulai menggunakan aplikasi di tempat Anda.

1

Unduh & Ekstrak

Unduh paket aplikasi nomor antrean melalui tombol download di atas. Setelah selesai, ekstrak file ZIP tersebut ke folder kerja di PC/Laptop Anda (misal ke C:\nomor-antrian). Anda juga dapat membaca Panduan Pengguna Lengkap (HTML) secara online.

2

Jalankan Server Lokal

Buka folder hasil ekstrak, kemudian jalankan file aplikasi server lokal (contoh: jalankan file start.bat atau ketik perintah node server.js di terminal). Server akan otomatis berjalan di port 3000.

3

Konfigurasi Shortcut Kiosk (Opsional)

Aplikasi akan otomatis mendeteksi browser Chrome/Edge dan membuat shortcut "Ambil Antrian" di Desktop. Jendela browser akan terbuka dalam mode Kiosk Fullscreen dengan parameter --kiosk-printing agar cetak thermal berjalan tanpa dialog.

4

Buka Mode Halaman

Buka browser dan arahkan ke alamat http://localhost:3000. Anda akan melihat halaman portal utama dan dapat membuka rute Kiosk (ambil tiket), Display (suara & layar TV), Loket (kontrol petugas), atau Admin.
